TraceAPI

FastAPI service for 3D neuron tracing with neuTube and Vaa3D APP2/smartTrace, returning SWC reconstructions and directional seed-based subtrees.

Markdown API reference · README

Run

Xvfb :99 -screen 0 1920x1080x24 -ac &
export DISPLAY=:99
export LD_LIBRARY_PATH="$PWD/algorithms:$LD_LIBRARY_PATH"
python -m uvicorn app:app --host 0.0.0.0 --port 8000 --workers 4

Endpoints

POST /trace_neutube
Run neuTube on an uploaded volume.
POST /trace_neutube_subtree
Run neuTube, then extract directional subtrees from seed pairs.
POST /trace_vaa3d_app2
Run iterative Vaa3D APP2 on an uploaded volume.
POST /trace_vaa3d_app2_subtree
Run APP2 from seed markers, then extract directional subtrees.
POST /trace_vaa3d_smartTrace
Run iterative Vaa3D smartTrace on an uploaded volume.

Seed Formats

Subtree routes accept one seed pair or multiple seed pairs. Single pair returns subtree.swc; multiple pairs return subtrees.zip.

s1=242.8440,249.8640,7.9980
s2=243.5728,248.0015,7.9980

s1=10,20,30;40,50,60
s2=11,20,30;42,50,60

s1=[[10,20,30],[40,50,60]]
s2=[[11,20,30],[42,50,60]]

Examples

neuTube Subtree

curl -X POST http://127.0.0.1:8000/trace_neutube_subtree \
  -F "tif_path=/data2/public_data/CWMBS/image/SN21.tif" \
  -F "s1=242.8440,249.8640,7.9980" \
  -F "s2=243.5728,248.0015,7.9980" \
  --output SN21_seed1_neuTube.swc

APP2 Subtree

curl -X POST http://127.0.0.1:8000/trace_vaa3d_app2_subtree \
  -F "tif_path=/data2/public_data/CWMBS/image/SN21.tif" \
  -F "s1=242.8440,249.8640,7.9980" \
  -F "s2=243.5728,248.0015,7.9980" \
  --output SN21_seed1_app2.swc

Subtree Algorithm

For each ordered pair s1 -> s2, TraceAPI finds the closest SWC node to s1, reroots the traced tree there, resamples with spacing equal to distance(s1, s2), keeps the root child branch closest to the target direction, and returns the resulting subtree.
